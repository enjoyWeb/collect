# 🚀 Chrome 自动化扩展 - 你好！这里是什么？

## 👋 简要说明

这是一个 **完全功能网络自动化 Chrome 扩展**，可以：

✅ 自动打开一列 URL  
✅ 在每个页面上自动点击元素（直观地选择，无需编码）  
✅ 自动关闭页面并继续下一个  
✅ 详细记录所有操作  

## ⚡ 3 分钟快速开始

### 1️⃣ 在 Chrome 中加载（1分钟）

```
打开 chrome://extensions/
  ↓
右上角开启"开发者模式"
  ↓
点击"加载已解压的扩展程序"
  ↓
选择：/Users/zhuyun/www/collect/
  ↓
✅ 完成！右上角出现扩展图标
```

### 2️⃣ 准备测试（1分钟）

```bash
# 打开终端，运行本地服务器
cd /Users/zhuyun/www/collect
python3 -m http.server 8000

# 然后在浏览器打开：
http://localhost:8000/test-page.html
```

### 3️⃣ 测试扩展（1分钟）

```
1. 点击 Chrome 右上角的扩展图标
2. 粘贴 URL：http://localhost:8000/test-page.html
3. 点击"🎯 选择元素"
4. 在打开的页面上点击任意元素
5. 看到蓝色高亮 + 提取的 CSS 选择器
6. 点击"✅ 确认选择"
7. ✨ 完成！现在可以运行自动化了
```

## 🤔 常见问题快速答案

**Q: 选择模式打开但点击没反应？**  
A: 刷新一下浏览器或等 1-2 秒让页面加载。

**Q: 看不到蓝色高亮？**  
A: 可能是网站 CSS 冲突。点击试试，选择器仍然能提取。

**Q: 如何运行自动化？**  
A: 配置好所有元素后，点击"▶️ 开始运行"按钮。

**Q: 数据会保存吗？**  
A: 是的，所有配置自动保存在 Chrome 本地存储。
