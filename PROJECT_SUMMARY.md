## 项目概述

**自动化采集工具**是一个功能完整的 Chrome 扩展，能够自动化地打开配置的 URL 链接，并在每个页面上执行指定的点击操作。

## 🚀 如何使用

### 1. 安装到 Chrome

```bash
1. 打开 Chrome 浏览器
2. 进入 chrome://extensions/
3. 启用右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 /Users/zhuyun/www/collect 文件夹
6. 完成！扩展将安装到 Chrome
```

### 2. 配置 URL

```bash
1. 点击 Chrome 工具栏的扩展图标
2. 输入要打开的 URL
3. 输入要点击的元素的 CSS 选择器
4. 设置页面加载延迟时间
5. 点击"➕ 添加链接"
6. 重复以上步骤添加更多链接
```

### 3. 运行自动化

```bash
1. 在"控制区"选择运行选项
2. 点击"▶️ 开始运行"
3. 扩展开始依次打开链接并点击元素
4. 观察"运行日志"中的操作记录
5. 完成后显示成功通知
```

### 4. 控制运行

```bash
暂停: ⏸️ 暂停按钮
继续: ▶️ 继续按钮
停止: ⏹️ 停止按钮
```

## 💻 技术架构

### 扩展组件

```
┌─────────────────────────────────────────┐
│       Chrome 浏览器                     │
├─────────────────────────────────────────┤
│                                         │
│  ┌──────────────┐  ┌──────────────┐   │
│  │  popup.html  │  │ popup.css    │   │
│  │  popup.js    │  │             │   │
│  │ (用户界面)   │  │             │   │
│  └──────┬───────┘  └──────────────┘   │
│         │                             │
│         │ 消息                         │
│         ▼                             │
│  ┌──────────────────────────────────┐ │
│  │    background.js                 │ │
│  │   (自动化核心逻辑)                 │ │
│  │   • 管理标签页                    │ │
│  │   • 控制流程                      │ │
│  │   • 通信协调                      │ │
│  └──────────┬───────────────────────┘ │
│             │                         │
│             │ 消息                     │
│             ▼                         │
│  ┌──────────────────────────────────┐ │
│  │   content-script.js              │ │
│  │   (页面注入脚本)                   │ │
│  │   • 查找元素                      │ │
│  │   • 模拟点击                      │ │
│  │   • 页面交互                      │ │
│  └──────────────────────────────────┘ │
│             │                         │
│             │ 点击                     │
│             ▼                         │
│   ┌──────────────────────┐           │
│   │   网页内容           │           │
│   │ (目标网站)           │           │
│   └──────────────────────┘           │
│                                         │
└─────────────────────────────────────────┘
```

### 数据流

```
用户配置 URL
    │
    ▼
保存到 Chrome Storage
    │
    ▼
点击开始按钮
    │
    ▼
popup.js → background.js
    │
    ▼
打开标签页 + content-script 注入
    │
    ▼
等待延迟时间
    │
    ▼
background → content-script
    │
    ▼
查找和点击元素
    │
    ▼
URL 处理完成
    │
    ▼
关闭标签页（可选）
    │
    ▼
打开下一个 URL 或完成
```
